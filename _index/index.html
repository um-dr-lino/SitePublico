Projetos

Automação da Leitura de Documentos com Python


Desenvolvi uma solução de automação para a extração de informações de documentos anexados em e-mails, utilizando a biblioteca Textract e a função Lambda. Esse processo otimiza o fluxo de trabalho ao reduzir o tempo e o esforço necessários para a leitura e processamento manual de documentos.

A automação realiza as seguintes etapas:
✅ Recebe e-mails com documentos anexados
✅ Utiliza o Textract para extrair texto dos anexos
✅ Manipula os dados extraídos
✅ Atribui automaticamente a campos específicos
✅ Armazena os anexos originais diretamente no SESuite para futura referência.

Essa solução melhora a eficiência operacional, eliminando tarefas manuais repetitivas e garantindo maior precisão na extração e organização das informações.
